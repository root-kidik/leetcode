> [!question] 
> Given an integerÂ `columnNumber`, returnÂ _its corresponding column title as it appears in an Excel sheet_.
> 
> For example:
> 
> A -> 1
B -> 2
C -> 3
...
Z -> 26
AA -> 27
AB -> 28 
... 

> [!example] 
> **Input:** columnNumber = 1
**Output:** "A" 

> [!example] 
> **Input:** columnNumber = 28
**Output:** "AB" 

> [!example] 
> **Input:** columnNumber = 701
**Output:** "ZY" 

> [!info] 
> - `1 <= columnNumber <= 231 - 1` 

___

> [!tip] 
> - `reverse`
> - `26`

___

```cpp
class Solution {
public:
    string convertToTitle(int columnNumber) {
        string buffer;

        while(columnNumber > 0)
        {
            columnNumber--;
            buffer.push_back(columnNumber % 26 + 'A');
            columnNumber /= 26;
        }

        reverse(buffer.begin(), buffer.end());
        return buffer;
    }
};
```
